# Product

## Register

brand

## Users

成年女性手作爱好者，主要在小红书获取灵感、买色卡、晒作品。她们用拼豆做谷子、做痛包、做立牌、做桌面装饰。

- **设备**：以手机端为主（晚间放松时段、地铁通勤），偶尔切到桌面端处理大尺寸图纸。
- **典型动作**：把小红书收藏的图 / 自己拍的爱豆照片 → 上传 → 自动出图纸 → 在线试拼 → 保存到作品馆。中间也有人不带图直接抽盲盒打发时间。
- **要做成的事**：从「看到一张想拼的图」到「拿到一份可执行的图纸 + 知道自己拼到哪」中间的所有摩擦都要消除。

## Product Purpose

线上拼豆工坊：上传图纸生成、色卡选择、试拼模拟、烫珠预览、作品收藏。一个让手作爱好者可以「不用先买豆子也能开始玩」的入口。

成功的样子：用户会把这个站加入收藏夹反复回访；晒图时主动带这个站名；身边朋友也来用。

## Brand Personality

**可爱、治愈、富足感。**

- **可爱**不是幼稚 — 是物体本身就讨喜（拼豆颗粒、色卡、马赛克），不是堆贴纸。
- **治愈**是节奏感 — 慢动作、不催促、留够呼吸；放松场景下使用，不要任何「您还有 3 步未完成」这种 SaaS 紧张感。
- **富足感**是关键 — 用户是来「逛色卡墙、看图鉴、被丰富的可能性激发」的，不是来看大留白的。信息可以密，但要整齐。

口吻：像一个比你早入坑两年、住你隔壁的女玩伴，不像客服也不像产品经理。

## Anti-references

明确不能像下面任何一种：

- **当前这版 shadcn/ui 默认 SaaS 模板**：rounded-lg + 中性灰 + uppercase tracking-wide 小标签 + "Bead Studio" 这种英文 eyebrow。一眼就是 AI 生成的。
- **二次元重色饱和 / 赛博朋克 / 霓虹激光**：任何"游戏宅"调性、外发光、辉光、粒子。
- **3D 圆胖购物车插画 / Notion AI 风万能贴画**：透明卷起人物、3D 渲染图标包，那种"AI 画的好看图"。
- **廉价女生粉**：粉狂 + Comic Sans + 心心圆圈 + 抓幼稚园点心。

## References

- **乐高官网 / Lego Build a Minifig**：拼搭玩具品牌的官方手法——大色块 hero、真实产品摄影占大版面、基于颗粒的几何母题、圆润现代字体、有玩心但精确。
- **宅不拼**：国内拼豆电商，色卡墙就是视觉本身，把材料密度当卖点。

共同点：**承认材料是真实的实物**，不抽象成"数字图块"；让"拼"这个动作的物理性渗透到 UI 里。

## Design Principles

1. **物质感优先（Material truth）**
   拼豆有形状、有摩擦、有反光、有色号。UI 必须承认这一点 — 不要把它当抽象数据可视化。允许颗粒看起来像颗粒，色卡看起来像色卡。

2. **拼搭即叙事（Build is the brand）**
   每个界面看起来都应该「像被拼出来」，而不是被设计出来。颗粒、网格、马赛克、色卡是核心母题，反复出现在分隔线、按钮、空状态、loading。

3. **富足而非繁杂（Abundance, not clutter）**
   充盈的色卡墙、密集的 trending 图鉴是对的；浮夸装饰、3D 插画堆叠、emoji 图标是错的。富足靠真实材料的多样性，不靠装饰。

4. **欢迎像走进店铺（Walk-in welcome）**
   首页第一秒是「你来啦」，不是「请选择您要进行的操作」。文案像主理人本人，不像系统。

5. **手作不是 lo-fi 的借口（Handmade is not lo-fi）**
   "温度"不等于"降低精度"。Lego 也很温暖，但每个倒角、每个颗粒间距都极其精确。我们要的是工艺感，不是粗糙感。

## Accessibility & Inclusion

- WCAG AA：正文对比 ≥4.5:1，图标 ≥3:1。
- **色卡不能只靠颜色识别**：色名 + 色号都要标（mard / coco / 等），保证色觉差异用户也能用。
- 移动端触摸目标 ≥44pt（颗粒画布除外，画布有专门的精度控制）。
- 尊重 prefers-reduced-motion：所有 playful 动效都要有降级路径。
- 中文字体在 16px 以上，行高 ≥1.6（中文密度高需要更喘息）。
