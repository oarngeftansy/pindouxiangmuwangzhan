# 拼豆板模式更新

## 更新内容

### 1. 新增拼豆板视图模式
添加了全新的拼豆板视图模式，让用户体验真实的在拼豆板上拼豆子的感觉。

**两种视图模式：**
- **简洁模式** - 原有的简洁界面，适合快速编辑和查看
- **拼豆板模式** - 逼真的拼豆板效果，仿佛在实体拼豆板上拼豆子

### 2. 拼豆板模式特性

#### 视觉效果
- **拼豆板背景**：浅米色 (#E8E4DD) 仿真拼豆板材质
- **立体钉子**：空位显示带有渐变和阴影的立体钉子
- **3D豆子效果**：
  - 圆形立体豆子，带有真实的高光和阴影
  - 内侧阴影模拟塑料材质
  - 外侧阴影模拟投影效果
  - 顶部高光增加立体感
  - 中心孔洞完整还原真实拼豆
- **细腻投影**：豆子在拼豆板上的投影效果
- **淡雅网格**：拼豆板模式下的淡色网格线

#### 交互体验
- 编辑画板标题旁边的视图模式切换按钮
- 实时切换，无需重新加载
- 保持所有原有编辑功能

### 3. 技术实现

#### 新增组件
- **PegboardCell.tsx** - 拼豆板单元格组件
  - 支持两种视图模式
  - 精细的CSS渐变和阴影效果
  - 响应式尺寸计算
  
- **ViewModeToggle.tsx** - 视图模式切换组件
  - 简洁的切换界面
  - 图标 + 文字标签
  - 响应式设计

#### 修改文件
- **BeadCanvas.tsx**
  - 添加 viewMode 状态管理
  - 集成 PegboardCell 组件
  - 动态背景色切换

### 4. 使用方法

1. 进入拼豆 DIY 编辑界面
2. 在"编辑画板"标题旁边找到视图模式切换按钮
3. 点击"拼豆板"图标切换到拼豆板模式
4. 点击"简洁"图标切换回简洁模式

### 5. 设计细节

#### 拼豆板颜色选择
- 背景色：#E8E4DD（浅米色，模拟真实拼豆板）
- 钉子：#C5BCB0 到 #A89F8F 渐变（金属质感）
- 网格线：rgba(180, 170, 155, 0.3)（淡色不干扰）

#### 豆子尺寸比例
- 钉子半径：beadSize * 0.12
- 豆子半径：beadSize * 0.42
- 孔洞半径：beadRadius * 0.35

这些比例经过精心调整，确保在不同缩放级别下都有良好的视觉效果。
