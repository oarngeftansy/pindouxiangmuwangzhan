# 拼豆板模式改进说明

## 更新内容

### 1. 默认视图模式调整
- **点击"开始拼豆"后默认显示拼豆板模式**
- 修改 `viewMode` 初始状态从 `'simple'` 改为 `'pegboard'`
- 简洁模式现在作为可切换的视图选项

### 2. 悬浮材料清单功能
- **在拼豆板模式下添加悬浮材料清单窗口**
- 材料清单可以自由拖动到任意位置
- 显示所有颜色的使用情况和拼豆进度
- 包含总进度条显示
- 可通过工具栏按钮切换显示/隐藏

### 3. 视图模式布局优化
- **拼豆板模式**：全屏主画布 + 可拖动悬浮参考窗 + 可拖动悬浮材料清单
- **简洁模式**：左右分屏布局（参考图纸 + 编辑画板）+ 右侧固定材料清单

### 4. 材料清单功能特性
- ✅ 显示各个颜色的色号
- ✅ 显示已拼/总数进度
- ✅ 已完成颜色显示绿色对勾
- ✅ 点击颜色可锁定并高亮对应区域
- ✅ 实时更新拼豆进度
- ✅ 总进度条展示整体完成情况

### 5. 工具栏新增按钮
- 在拼豆板模式下新增"材料清单"显示/隐藏按钮
- 使用绿色主题色与参考图纸的蓝色区分
- 只在拼豆板模式下显示，简洁模式下材料清单固定在右侧

### 6. 悬浮窗交互优化
- 参考图纸悬浮窗：蓝紫色主题，左侧默认位置
- 材料清单悬浮窗：绿色主题，右侧默认位置
- 两个悬浮窗都支持拖动调整位置
- 参考图纸窗口支持 +/- 缩放功能
- 点击关闭按钮可隐藏悬浮窗

## 使用流程

1. **图片转换完成后** → 点击"开始拼豆" → **自动进入拼豆板模式**
2. **拼豆板模式下**：
   - 左侧悬浮窗显示参考图纸（可拖动、缩放、关闭）
   - 右侧悬浮窗显示材料清单（可拖动、关闭）
   - 中央全屏显示拼豆创作台
3. **切换到简洁模式**：
   - 点击工具栏的"视图模式"切换按钮
   - 左右分屏显示参考图纸和编辑画板
   - 材料清单固定在右侧

## 技术实现

### 新增状态管理
```typescript
// 悬浮材料清单相关状态
const [floatingMaterialPosition, setFloatingMaterialPosition] = useState({ x: 0, y: 100 });
const [isDraggingMaterial, setIsDraggingMaterial] = useState(false);
const [materialDragOffset, setMaterialDragOffset] = useState({ x: 0, y: 0 });
const [showMaterialList, setShowMaterialList] = useState(true);
```

### 悬浮窗拖动实现
- 使用 `onMouseDown` 捕获拖动开始
- 监听 `mousemove` 更新位置
- 监听 `mouseup` 结束拖动
- `e.stopPropagation()` 防止拖动时触发其他事件

### 响应式布局
- 拼豆板模式：移除 grid 布局，使用 relative 定位
- 简洁模式：保持 `lg:grid-cols-[1fr_320px]` 布局
- 悬浮窗使用 `fixed` 定位，不影响主画布布局

## 用户体验提升

1. **更专注的创作体验**：拼豆板模式下的全屏画布让创作更沉浸
2. **灵活的参考方式**：可自由调整参考窗和材料清单位置，适应不同习惯
3. **实时进度反馈**：随时查看各颜色完成情况，掌握创作进度
4. **无缝模式切换**：随时在拼豆板模式和简洁模式间切换
